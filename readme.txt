Nombre del proyecto: Prediccion de valores de los indices de precios de consumidores 1.0.0

Funcionamiento:
Predecir los índices de los precios de los consumidores negociables y no negociables en septiembre de 2023 usando el algoritmo de aprendizaje automatico Regresión Lineal.
Pasos:
1. Leer el conjunto de datos original
2. Limpiar el conjunto de datos (valores nulos, valores atípicos, etc.)
3. Guardar y cargar el conjunto de datos limpio
4. Comprobar la correlacion entre las variables numéricas
5. Si las variables no están correlacionadas entre sí, se empieza a crear el modelo
6. Dividir el conjunto de entrenamiento y el conjunto de test
7. Preprocesar conjunto de entrenamiento (variables numéricas y categóricas)
8. Entrenar modelo de regresión lineal
9. Predecir resultados del modelo
10. Guardar resultados del modelo en un nuevo conjunto de datos
11. Graficar resultados

Técnicas utilizadas:
-Limpieza de datos
-Análisis de datos
-Aprendizaje automático
-Visualización de datos

Conjunto de datos utilizado: consumers-prices-indexes-september-2023-quarter-tradables-and-non-tradables
Link: https://www.stats.govt.nz/assets/Uploads/Consumers-price-index/Consumers-price-index-September-2023-quarter/Download-data/consumers-price-index-september-2023-quarter-tradables-and-non-tradables.csv

Tecnologías utilizadas:
-Lenguaje de programación Python==3.12.0
-Entorno de desarrollo integrado PyCharm
-Editor de texto Visual Studio Code
*Las librerías de Python utilizadas en el proyecto se encuentran en el archivo requeriments


Project name: Consumers prices indexes values prediction 1.0.0

Functioning:
Predict tradables and non tradables consumers prices indexes in september 2023 using the Linear Regression machine learning algorithm.
Steps:
1. Read original dataset
2. Clean dataset
3. Save and load new clean dataset
4. Verify variable correlation
5. If the variables are not correlated with each other, the model begins to be created
6. Split the train set and test set
7. Preprocess train set (numeric variables and categoric variables)
8. Train linear regression model
9. Predict model results
10. Savi model reults on a new dataset
11. Graph results
